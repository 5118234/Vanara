# TaskDialog.ExpandFooterArea Property 
 

Indicates that the string specified by the ExpandedInformation member should be displayed at the bottom of the dialog’s footer area instead of immediately after the dialog’s content. This flag is ignored if the ExpandedInformation member is null.
