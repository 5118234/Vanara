Type: SystemString
Pointer to the null-terminated Unicode string name of the disk file to be examined. The pwcsName parameter is passed uninterpreted to the underlying file system.

Return Value

Type: HRESULT
S_OKIndicates that the file contains a storage object.S_FALSEIndicates that the file does not contain a storage object.STG_E_FILENOTFOUNDIndicates that the file was not found.
