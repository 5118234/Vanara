Methods
  NameDescription
COMBINE_PAGE_ACTIVATION
Combines the AclUISI_PAGE_TYPE and AclUISI_PAGE_ACTIVATED types for use in the last parameter of EditSecurityAdvanced(IntPtr, AclUIISecurityInformation, UInt32) method.
EditSecurity
The EditSecurity function displays a property sheet that contains a basic security property page. This property page enables the user to view and edit the access rights allowed or denied by the ACEs in an object's DACL.
EditSecurityAdvanced(IntPtr, AclUIISecurityInformation, UInt32)
The EditSecurityAdvanced function extends the EditSecurity function to include the security page type when displaying the property sheet that contains a basic security property page. This property page enables the user to view and edit the access rights allowed or denied by the access control entries (ACEs) in an object's discretionary access control list (DACL).
