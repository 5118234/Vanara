DrawThemeTextOptions Properties

The UxThemeDrawThemeTextOptions type exposes the following members.

Properties
  NameDescription
AlternateColor
Gets or sets a value that specifies an alternate color property to use when drawing text.
AlternateFont
Gets or sets an alternate font property to use when drawing text.
AntiAliasedAlpha
Gets or sets a value indicating whether to draw text with antialiased alpha. Use of this flag requires a top-down DIB section. This flag works only if the HDC passed to function DrawThemeTextEx has a top-down DIB section currently selected in it. For more information, see Device-Independent Bitmaps.
ApplyOverlay
Gets or sets a value indicating whether text will be drawn on top of the shadow and outline effects ( true) or if just the shadow and outline effects will be drawn ( false).
BorderColor
Gets or sets the color of the outline that will be drawn around the text.
BorderSize
Gets or sets the radius of the outline that will be drawn around the text.
Callback
Gets or sets the callback function.
Default
Gets an instance with default values set.
GlowSize
Gets or sets the size of a glow that will be drawn on the background prior to any text being drawn.
LParam
Gets or sets the parameter for callback back function specified by Callback.
ReturnCalculatedRectangle
Gets or sets a value indicating whether the pRect parameter of the DrawThemeTextEx(UxThemeSafeThemeHandle, Gdi32SafeDCHandle, Int32, Int32, String, Int32, DrawTextFlags, RECT, UxThemeDrawThemeTextOptions) function that uses this structure will be used as both an in and an out parameter. After the function returns, the pRect parameter will contain the rectangle that corresponds to the region calculated to be drawn.
ShadowColor
Gets or sets the color of the shadow drawn behind the text.
ShadowOffset
Gets or sets the amount of offset, in logical coordinates, between the shadow and the text.
ShadowType
Gets or sets the type of the shadow that will be drawn behind the text.
TextColor
Gets or sets the color of the text that will be drawn.
