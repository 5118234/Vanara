ComCtl32ImageList_DrawEx Method

Draws an image list item In the specified device context. The function uses the specified drawing style and blends the image with the specified color.

The y-coordinate at which to draw within the specified device context.
dx
Type: SystemInt32
The width of the portion of the image to draw relative to the upper-left corner of the image. If dx and dy are zero, the function draws the entire image. The function does not ensure that the parameters are valid.
dy
Type: SystemInt32
The height of the portion of the image to draw, relative to the upper-left corner of the image. If dx and dy are zero, the function draws the entire image. The function does not ensure that the parameters are valid.
rgbBk
Type: SystemUInt32
The background color of the image. This parameter can be an application-defined RGB value or one of the following values: CLR_NONE or CLR_DEFAULT.
rgbFg
Type: SystemUInt32
The foreground color of the image. This parameter can be an application-defined RGB value or one of the following values: CLR_NONE or CLR_DEFAULT.
fStyle
Type: Vanara.PInvokeComCtl32IMAGELISTDRAWFLAGS
The drawing style and, optionally, the overlay image. For information about specifying an overlay image index, see the comments section at the end of this topic. This parameter can be a combination of an overlay image index and one or more of the ComCtl32IMAGELISTDRAWFLAGS values.

Return Value

Type: Boolean
Returns nonzero if successful, or zero otherwise.
