dwSig
Type: Vanara.PInvokeShell32ShellDataBlockSignature
The data block's signature. The signature value for a particular type of data block can be found in its structure reference. For a list of supported data block types and their associated structures, see IShellLinkDataList.

Return Value

Type: SafeLocalHandle
The address of a pointer to a copy of the data block structure. If IShellLinkDataList::CopyDataBlock returns a successful result, the calling application must free the memory when it is no longer needed by calling LocalFree.
