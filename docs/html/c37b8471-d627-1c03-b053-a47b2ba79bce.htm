Parameters

rwDepth
Type: SystemUInt32

Indicates the number of stores, beginning with the child, of the backing store chain to open as read/write. The remaining stores in the differencing chain will be opened read-only. This is necessary for merge operations to succeed.

ValueMeaning
0Do not open for read/write at any depth. This value should be used for read-only operations.
OPEN_VIRTUAL_DISK_RW_DEPTH_DEFAULT (1)Default value to use if no other value is desired.
n (user-defined)This integer value should be the number of merge levels plus one, if a merge operation is intended.
