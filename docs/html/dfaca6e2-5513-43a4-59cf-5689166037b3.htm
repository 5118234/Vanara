NetListMgrINetworkCostManagerGetCost Method

The GetCost method retrieves the current cost of either a machine-wide internet connection, or the first-hop of routing to a specific destination on a connection. If destIPaddr is NULL, this method instead returns the cost of the network used for machine-wide Internet connectivity.

Parameters

pCost
Type: Vanara.PInvokeNetListMgrNLM_CONNECTION_COST
A DWORD value that indicates the cost of the connection. The lowest 16 bits represent the cost level, and the highest 16 bits represent the flags. Possible values are defined by the NetListMgrNLM_CONNECTION_COST enumeration.
pDestIPAddr
Type: SystemIntPtr
An NetListMgrNLM_SOCKADDR structure containing the destination IPv4/IPv6 address. If NULL, this method will instead return the cost associated with the preferred connection used for machine Internet connectivity.
