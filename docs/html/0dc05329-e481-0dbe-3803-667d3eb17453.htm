AdvApi32LsaOpenPolicy Method (AdvApi32LsaPolicyRights, String)

The LsaOpenPolicy function opens a handle to the Policy object on a local or remote system. You must run the process "As Administrator" so that the call doesn't fail with ERROR_ACCESS_DENIED.

Parameters

DesiredAccess
Type: Vanara.PInvokeAdvApi32LsaPolicyRights
An ACCESS_MASK that specifies the requested access rights. The function fails if the DACL of the target system does not allow the caller the requested access. To determine the access rights that you need, see the documentation for the LSA functions with which you want to use the policy handle.
SystemName (Optional)
Type: SystemString
Name of the target system. The name can have the form "ComputerName" or "\\ComputerName". If this parameter is NULL, the function opens the Policy object on the local system.

Return Value

Type: AdvApi32SafeLsaPolicyHandle
A pointer to an LSA_HANDLE variable that receives a handle to the Policy object.
