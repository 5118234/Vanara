Methods

Clears all destination entries from the Recent and Frequent categories in an application's Jump List.

Removes a single destination from the Recent and Frequent categories in a Jump List.
SetAppID
Specifies a unique AppUserModelID for the application from whose taskbar button's Jump List the methods of this interface will remove destinations. This method is optional.
