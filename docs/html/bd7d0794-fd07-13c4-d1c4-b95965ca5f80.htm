SI_INHERIT_TYPE Fields

The AclUISI_INHERIT_TYPE type exposes the following members.

Fields
  NameDescription
dwFlags
A set of AclUISI_INHERIT_Flags that indicate the types of ACEs that can be inherited by the ChildObjectTypeId. These flags correspond to the AceFlags member of an ACE_HEADER structure.
pguid
A pointer to a GUID structure that identifies the type of child object. This member can be a pointer to GUID_NULL. The GUID corresponds to the InheritedObjectType member of an object-specific ACE.
pszName
A display string that describes the child object.
