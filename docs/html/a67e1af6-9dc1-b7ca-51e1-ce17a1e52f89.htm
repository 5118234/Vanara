INetworkListManager Methods

The NetListMgrINetworkListManager type exposes the following members.

Methods
  NameDescription
ClearSimulatedProfileInfo
Clears the connection profile values previously applied to the internet connection profile by SetSimulatedProfileInfo. The next internet connection query, via GetInternetConnectionProfile, will use system information.
GetConnectivity
Returns the connectivity state of all the networks on a machine.
GetNetwork
Retrieves a network based on a supplied Network ID.
GetNetworkConnection
Retrieves a network based on a supplied Network Connection ID.
GetNetworkConnections
Gets an enumerator that contains a complete list of the network connections that have been made.
GetNetworks
Retrieves networks based on the supplied Network IDs.
SetSimulatedProfileInfo
Applies a specific set of connection profile values to the internet connection profile in support of the simulation of specific metered internet connection conditions.

The simulation only applies in an RDP Child Session and does not affect the primary user session. The simulated internet connection profile is returned via the Windows Runtime API GetInternetConnectionProfile.
