Data
Gets or sets the data for the input dialog box. The data type will determine the type of input mechanism displayed. For simple types, a TextBox with validation, or a CheckBox or a ComboBox will be displayed. For classes and structures, all of the public, top-level, fields and properties will have input mechanisms shown for each. See Remarks for more detail.
