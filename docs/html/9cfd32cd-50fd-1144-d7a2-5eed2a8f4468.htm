Parameters

netServerEnumFilter (Optional)
Type: Vanara.PInvokeNetApi32NetServerEnumFilter
A value that filters the server entries to return from the enumeration.
domain (Optional)
Type: SystemString
A string that specifies the name of the domain for which a list of servers is to be returned. The domain name must be a NetBIOS domain name (for example, Microsoft). The NetServerEnum function does not support DNS-style names (for example, microsoft.com). If this parameter is NULL, the primary domain is implied.
level (Optional)
Type: SystemInt32
The information level of the data requested. If this value is 0, then the method will extract all digits to form the level (e.g. SERVER_INFO_101 produces 101).

Type Parameters

T
The type of the structure to have filled in for each server. This must be SERVER_INFO_100 or SERVER_INFO_101.

Return Value

Type: IEnumerableT
A managed array of the requested type.
