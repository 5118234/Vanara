LsaHandle
Type: Vanara.PInvokeNetSecApiSafeLsaConnectionHandle
Handle obtained from a previous call to LsaRegisterLogonProcess or LsaConnectUntrusted.
PackageName
Type: SystemString
A string that specifies the name of the authentication package. The package name must not exceed 127 bytes in length. The following table lists the names of the Microsoft-provided authentication packages.
ValueMeaning
MSV1_0_PACKAGE_NAMEThe MSV1_0 authentication package name.
MICROSOFT_KERBEROS_NAMEThe Kerberos authentication package name.
NEGOSSP_NAMEThe Negotiate authentication package name.
AuthenticationPackage
Type: SystemInt32
Pointer to a ULONG that receives the authentication package identifier.

Return Value

Type: UInt32
If the function succeeds, the return value is STATUS_SUCCESS.

If the function fails, the return value is an NTSTATUS code. The following are possible error codes.

Return codeDescription
STATUS_NO_SUCH_PACKAGEThe specified authentication package is unknown to the LSA.
STATUS_NAME_TOO_LONGThe authentication package name exceeds 127 bytes.
