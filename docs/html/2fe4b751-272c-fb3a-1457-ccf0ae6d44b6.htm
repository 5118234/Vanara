MAX_PREFERRED_LENGTH Field

NetApi32MAX_PREFERRED_LENGTH Field

A constant of type DWORD that is set to –1. This value is valid as an input parameter to any method that takes a PreferedMaximumLength parameter. When specified as an input parameter, this value indicates that the method MUST allocate as much space as the data requires.
